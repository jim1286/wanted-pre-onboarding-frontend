# 제목

원티드 프리온보딩 프론트엔드 - 선발 과제

## 설명

원티드 프리온보딩 선발 과제이며, 로그인/회원가입 구현 및 Todo List를 구현했습니다.

### 설치

```
npm install
```

### 실행

cd wanted-pre-onboarding-frontend

```
npm start
```

## 사용 라이브러리

- axios - "^1.2.1"
- react-router-dom - "^6.4.5"
- react-hook-form - "^7.40.0"
- styled-components - "^5.3.6"
